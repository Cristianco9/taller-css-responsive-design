Modelo Relacional

El modelo relacional organiza los datos en tablas (relaciones) que se conectan mediante llaves primarias y foráneas.

SQL

Structured Query Language (SQL) es el lenguaje más usado para gestionar y consultar bases de datos relacionales.

NoSQL

Las bases de datos NoSQL son flexibles y permiten manejar documentos, grafos, claves-valor o columnas en gran escala.
